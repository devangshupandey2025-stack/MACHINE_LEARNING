# Machine Learning - Decision Trees

Implementation of Decision Tree classifiers using scikit-learn.

## Files

- **siu1.py** - Basic Decision Tree model that predicts pass/fail based on study hours using `sklearn.tree.DecisionTreeClassifier`. Trained on 8 samples (1-8 hours), predicts for 2.5 hours.

- **siu2.py** - Visualizes the trained Decision Tree using `sklearn.tree.plot_tree` with matplotlib, showing feature names ("Hours") and class labels ("Fail", "Pass").

## Requirements

- numpy
- scikit-learn
- matplotlib (for siu2.py)

## Usage

```bash
python siu1.py
python siu2.py
```
